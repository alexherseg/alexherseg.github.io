makeoveR monday 2018

What is Makeover Monday?

Makeover Monday is a weekly data visualization series that allows you to re-work/re-imagine a chart. It is also a good way for me to practice my plotting skills in R. I really like this project, and I cant believe I didnt find it sooner (I didnt get started until week 06, and Ive since gone back and completed weeks 05 and 04). I dont know if Ill be able to, but Im going to try my best to complete all 52 weeks.

R

So far Ive used the ggplot2 package to make all of the plots, but I plan on using base R too. I also want to make some interactive charts and put them on shinyapps.io. I usually get lazy and I dont do it, but it is one of my goals.

Visualizations

For each graph below, I include some short comments and the R code needed to make the chart. My code is often not optimal, but you can see (I hope) what my though process was when creating each graph. I probably should customize my own theme for ggplot (I use a lot of the same options in each plot), but it is easier for people to reproduce the plot if they can copy the code straight from the html.
